# ProjetoBanco

Este projeto foi feito em acompanhamento da aula de C# Foursys;

Conceito do Projeto: Este projeto tem por finalidade a possibilidade de que o usuário possa realizar um cadastro bancário, onde também estará incluida, a possibilidade de realizar transferencias, depositos e consulta de conta, também foi integrado a criação de conta corrente e poupança, podendo assim, ser realizados calculos de juros e taxas de rendimentos, também é possivel a realização de cadastros de chaves pix, atravez da integração de interfaces de leitura e gravação para uma planilha excel .csv;

Estrutura do projeto: Orientado a objeto, heranças, interfaces (Gravação e Leitura); Classes especificas utilizadas: StreamWritter, StreamReader, Exceptions e Enuns.
